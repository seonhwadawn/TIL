# This is a H1 State란 무엇인가?

- setState()는 컴포넌트의state에 객체에 대한 업데이트를 실행한다.
- state가변경되면 컴포넌트를 리렌더링 된다
- 개별 상태를 가지게 된다 => 컴포넌트별로 별개로 인식한다.
- State는 컴포넌트 인스턴스 기반단위로 나누어져있다 명심할것!
